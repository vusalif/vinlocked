CountryOdyssey

Documentation

1. Introduction

CountryOdyssey is a fun and educational geography game where players can test and improve their knowledge of world countries by locating them on a map. With different levels of difficulty (Easy, Normal, Hard), the game is perfect for anyone who loves geography and wants to challenge themselves or compete with friends.

2. Game Overview

In CountryOdyssey, players will be given the name of a country and must click on the map to correctly identify its location. The game has two modes:

  • Single-player: Players can play by themselves and aim for the highest score.
  • Multiplayer: Players can compete against friends or other online players to find countries faster and more accurately.

3. How to Play

Single-player Mode

  1. Start the game by choosing Single-player.
  2. You will be shown the name of a country.
  3. Click on the map where you believe that country is located.
  4. The game will give you feedback on your selection.
  5. Try to finish as many countries as possible within the time limit.

Multiplayer Mode

  1. Choose Multiplayer from the main menu.
  2. Select whether you want to play with friends or random players.
  3. Once the game starts, each player will take turns locating countries.
  4. Points will be awarded based on accuracy and speed.
  5. The player with the highest score at the end wins the round.

4. Game Features

Difficulty Levels

  • Easy: Well-known locations and large countries
  • Normal: Moderate challenge with intermediate countries
  • Hard: Less well-known countries in difficult-to-identify regions

5. Technical Details

CountryOdyssey is built using modern web technologies to ensure smooth gameplay and responsive design across all devices.

But it is highly recommended to play it on a desktop or laptop because it is optimized for them.

The game is built using HTML, CSS, and JavaScript. It uses the Leaflet library for the map interface and the OpenStreetMap API for the map data.

6. Troubleshooting

  • Game not loading: Try refreshing the page or clearing browser cache
  • Multiplayer issues: Check your internet connection
  • Game is lagging: Close other tabs or apps using resources
  • Contact with us. Feedback & Suggestions & Report Bugs
